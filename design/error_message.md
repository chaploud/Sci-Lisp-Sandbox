# Error Messages

## Python Example

```bash
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
NameError: name 'hoge' is not defined
```

- Trace: Debug Modeではまた違った感じかなぁ
- File: <stdin>
- line: 1
- scope: <module>
- Error: NameError
- Message: name 'hoge' is not defined

- 必要最低限はこれ!
- 行番号、列番号は後々
- 優しい表示も後々
